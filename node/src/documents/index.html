Semantic UI

UI is the DNA of the web.

Simple

Design pages by describing them.
Writing pages in Semantic is as easy as writing english sentences.

Lean

Use only the bits you want, leave the rest.
Every UI element in semantic is written to be stand-alone, or loosely coupled. Use only what you need.

Helpful

Not happy with off the shelf plugins?
Semantic modules are written to be easy to read and extensible. Every plugin provides out of the box debug tools so you can monitor performance without breakpoints.
