Sidebar

A sidebar is a menu hidden beside page content

Initializing Usage Types Behavior Settings

Initializing

Sidebars are initialized by calling sidebar on an element with class ui sidebar

$('.ui.sidebar') .sidebar() ;

Controlling a sidebar

It may also be useful to set a dedicated button as controlling a sidebar. You can do this by calling a separate method

$('.sidebar') .sidebar('attach events', '.toggle.button') ;

Altering a sidebar width

Altering the size of sidebar can be done by modifying sidebar.css

Usage

If a sidebar is called without any arguments all default settings will be used.

Visibility

After the animation queue finishes for an element, its final visibility state is determined. If an animation is an outward sidebar, the final visibility status will be hidden. If an animation is inward the element will be visible after the animatino finishes.

$('.dog.image') .sidebar('scale') ;

Static Animations

Animations that do not have an in or out animation defined, will maintain their current visibility after running

$('.dog.image') .sidebar('pulse') ;

Sidebar Direction

If an animation direction is not specified it will automatically be determined based on the elements current visibility. For example, if the element is visible the animation "fade out" will be called, if the animation is not visible "fade in".

$('.dog.image') // fade up out is used if available .sidebar('fade up') // this is now fade up in .sidebar('fade up') ;
$('.dog.image') // if a direction if specified it will be obeyed .sidebar('horizontal flip in') .sidebar('vertical flip in') ;

Passing in settings

Sidebars use similar argument shorthand as animate. You can specify callback functions, animation duration, and other settings using the same arguments. Durations can be specified as strings with css shorthand, or with numbers.

Queuing animations

Animations called in sequence will be queued. Any queued animation will automatically determine the sidebar direction if none is specified.

$('.dog.image') .sidebar('horizontal flip', '500ms') .sidebar('horizontal flip', 500, function() { alert('done!'); }) ;

Types

Emphasis

Flash

An element can flash to draw attention to itself

$('.dog.image') .sidebar('flash') ;

Shake

An element can shake to draw attention to its position

$('.dog.image') .sidebar('shake') ;

Tada

An element can give user positive feedback for an action

$('.dog.image') .sidebar('tada') ;

Bounce

An element can bounce to politely remind you of itself

$('.dog.image') .sidebar('bounce') ;

Appearance

Flip

An element can flip into or out of view vertically or horizontally

$('.dog.image') .sidebar('horizontal flip') .sidebar('vertical flip') ;

Fade

An element can fade into or out of view descending and ascending

$('.dog.image') .sidebar('fade') .sidebar('fade up') .sidebar('fade down') ;

Scale

An element can scale into or out of view

$('.dog.image') .sidebar('scale') ;

Slide

$('.dog.image') .sidebar('slide up') .sidebar('slide down') ;
