Dimmer
Dimmers focus a users attention on particular content

Usage Types Behavior States Variations

Usage

Initializing a dimmer

// if you initialize a dimmer on a dimmable section, a dimmer will automatically be created inside $('.ui.dimmable') .dimmer() ; // if you initialize a dimmer directly, it will dim its current parent $('.ui.dimmer') .dimmer() ;

Types

Dimmable

A dimmable is a section of content that can be dimmed

A dimmer will automatically be inserted the first time you initialize a dimmer on a dimmable section

Dimmer with Content

A dimmer can display simple messages in a content block

Messages that are direct descendants of a content block will be vertically centered table cell. To avoid content being stretched include it as child element.

Behavior

Display

You can display a dimmer by either invoking dimmer show on a section or a dimmer itself. If you choose to dim a dimmable section, a dimmer will automatically be created.

// these two are the same $('.ui.dimmable) .dimmer('show') ; $('.ui.dimmable .dimmer') .dimmer('show') ; // these two as well $('.ui.dimmable) .dimmer('hide') ; $('.ui.dimmable dimmer') .dimmer('hide') ;

States

Active

An active dimmer will dim its parent container

Disabled

A disabled dimmer cannot be activated

Variations

Simple Dimmer

A dimmer can be controlled without javascript

Having any parent element receive the class "dimmed" will trigger the dimmer to display.
